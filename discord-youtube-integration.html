YouTube + Discord automation features

Turn your Discord server into the operational hub for your YouTube channel.

Upload notifications

New video goes live, Discord gets notified instantly. Rich embeds with thumbnail, title, duration, and direct link. Supports multiple channels and custom formatting per channel.

Subscriber verification

Members authenticate via Google OAuth. The bot confirms they subscribe to your channel and assigns a verified role. Periodic re-checks catch unsubscribes automatically.

Membership gating

Gate Discord channels behind active YouTube channel memberships. When a membership lapses, access revokes automatically. Tiered memberships map to tiered Discord roles.

Watch parties and premieres

Schedule watch parties with countdown timers. Auto-create voice channels, ping interested members, and post synchronized timestamps so everyone watches together.

Analytics feeds

Post daily or weekly performance summaries to a staff channel. Views, watch time, subscriber delta, top-performing videos, and comment sentiment at a glance.

Video search and lookup

Slash command to search your channel's catalog from Discord. Members find relevant content without leaving the server. Useful for educational and tutorial channels.

Common YouTube + Discord workflows

  • Creator community hub: new upload triggers a notification embed, members discuss in a dedicated thread, bot pins top comments and cross-posts highlights
  • Subscriber-only perks: verify subscription status via OAuth, assign "Subscriber" role, unlock exclusive channels, voice rooms, and bot commands
  • YouTube Membership to Discord access: members who pay for channel memberships automatically receive corresponding Discord roles with premium content access
  • Premiere countdown: bot announces scheduled premieres 24h, 1h, and 5min before. Creates a temporary voice channel for live reactions. Cleans up after the event
  • Content feedback pipeline: post each new video to a feedback channel. Members react with structured ratings. Bot aggregates scores and posts a summary for the creator

How the YouTube API integration works

The YouTube Data API v3 and Google OAuth provide the building blocks for deep Discord integration.

  • YouTube Data API v3 (Channels): fetches subscriber count, video count, total views, and channel metadata. Powers analytics feeds and profile displays.
  • YouTube Data API v3 (Search/Videos): lists recent uploads, searches video catalogs, and retrieves video details like duration, view count, and publish date. Drives notification and search features.
  • YouTube Data API v3 (Subscriptions): verifies whether an authenticated user subscribes to a specific channel. The core of subscriber verification workflows.
  • Google OAuth 2.0: secure authentication flow where members grant read-only access to their subscription and membership data. No passwords shared, revocable at any time.
  • PubSubHubbub (WebSub): real-time push notifications when a new video is published. Eliminates polling and ensures instant upload alerts with minimal latency.
